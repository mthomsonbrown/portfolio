This Very Portfolio

Everyone needs a portfolio! There's a discrepancy between the things I tinker with at home and the work I've done on the job, so I thought it would be good to have a place to showcase some of the avenues I've explored.

  • I chose Angular for this project so that I have the option to add some dynamic content later on.
  • I'm using Angular Material as the UI framework. It's a good framework, but I keep thinking things would be more straight-forward with Bootstrap. I'm sticking with it for a bit anyway. I assume my preference for Bootstrap is due to my lack of experience with vanilla html/css. Still I think Bootstrap was more responsive out of the box.
